HTML Colors

HTML Colors

HTML colors are specified with predefined color names, or with RGB, HEX, HSL, RGBA, or HSLA values.

Color Names

In HTML, a color can be specified by using a color name:

      
Tomato

      
Orange

      
DodgerBlue

      
MediumSeaGreen

      
Gray

      
SlateBlue

      
Violet

      
LightGray
Text Color

You can set the color of text

code : 
    
      <h3 style="color:Tomato;">“Wonder is the beginning of wisdom.”</h3>
      <p style="color:DodgerBlue;">“Whatever you are, be a good one.”.</p>
      <p style="color:MediumSeaGreen;">“The only true wisdom is in knowing you know nothing.”</p>
output : 
      
“Wonder is the beginning of wisdom.”

      
“Whatever you are, be a good one.”

      
“The only true wisdom is in knowing you know nothing.”
Background Color

You can set the background color for HTML elements

code : 
    
<h1 style="background-color:DodgerBlue;">"Step out of my sunlight.""</h1>
<p style="background-color:Tomato;">“People who know little are usually great talkers”
</p>
output : 
		
"Step out of my sunlight.""

	 	
“People who know little are usually great talkers, while men who know much say little.”
Border Color

You can set the color of borders

code : 
    
		<h1 style="border:2px solid Tomato;">Hello World</h1>
		<h1 style="border:2px solid DodgerBlue;">Hello World</h1>
		<h1 style="border:2px solid Violet;">Hello World</h1>
output : 
		
Hello World
